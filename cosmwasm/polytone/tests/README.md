To run IBC tests (you will need [just
installed](https://just.systems/man/en/installation.html)):

```
just simtest
```

This contract uses the Cosmos SDK's
[simulator](https://docs.cosmos.network/sdk/latest/learn/concepts/baseapp#:~:text=is%20never%20committed.-,Simulate,-(ExecModeSimulate)%3A%20runs) to test
IBC interactions between chains.
